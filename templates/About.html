📝 About My Todo

A simple, fast way to keep track of what matters.

What is My Todo?

My Todo is a lightweight task management web app built with Flask (Python) on the backend and Bootstrap for a clean, responsive interface. It helps you add, organize, and track your daily tasks so nothing slips through the cracks.

How to use it

1

Create an account / Log in

Sign up with your email or log in if you already have an account.

2

Add a task

Use the input box on your dashboard to type a task and click "Add".

4

Edit or delete tasks

Use the edit (pencil) icon to update a task, or the trash icon to remove it.

5

Stay organized

Your tasks are saved to your account, so you can log in from anywhere and pick up where you left off.

Built with

Python Flask Bootstrap SQLite / SQLAlchemy
